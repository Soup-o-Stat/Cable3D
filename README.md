# CABLE3D

Cable3D is a spline-based procedural 3D cable generator designed for flexible scene creation and fast level design. It allows you to generate cables, wires, ropes, and other curved structures directly along a Path3D spline with full control over shape and behavior.
The system supports multiple preset shapes, including straight, hanging, zigzag, arc, and circular coil forms, making it suitable not only for cables but also for decorative or technical procedural elements.
Key features include mesh instancing via MultiMesh for high performance, automatic alignment to curve direction, adjustable sag for realistic hanging behavior, twist control along the spline, random offsets and rotations for natural variation, and corner smoothing for eliminating harsh angle transitions.
Cable3D is designed as a general-purpose PCG tool for any spline-driven geometry, from simple wires and pipes to complex environmental details and stylized procedural assets.
